Crow
====

## About
Crow is a nodejs/mongodb app to track contacts and (eventually) to send out newsletters.

## Dependencies
- Express
- Mongodb
- Jade
- Stylus

## How to Use
1.  Make sure mongodb is running locally
2. `git clone https://github.com/notcoffeetable/Crow.git`
3. `cd Crow`
4. `node app`
